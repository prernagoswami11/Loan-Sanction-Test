# Loan-Sanction-Test
 Exploratory Data Analysis (EDA) on a Loan Sanction dataset using Python, Pandas, Matplotlib, and Seaborn. The project focuses on data cleaning, handling missing values, and analyzing applicant income, credit history, and demographic patterns to understand factors related to loan applications

Loan Sanction EDA Analysis

This project performs Exploratory Data Analysis (EDA) on a loan sanction dataset to understand applicant financial profiles and factors that may affect loan approval.

📂 Dataset Overview

The dataset contains 367 loan applicants with features such as:

Gender

Marital Status

Dependents

Education

Employment Status

Applicant & Co-applicant Income

Loan Amount

Loan Term

Credit History

Property Area

🔧 Steps Performed

Data loading using Pandas

Handling missing values

Data cleaning and transformation

Identifying numerical and categorical features

Visual analysis using Seaborn and Matplotlib

📊 Key Findings

Majority of applicants are male and married.

Urban areas contribute the most loan applicants.

Most applicants are not self-employed.

Applicants with zero dependents have stronger credit history.

Higher dependents correlate with higher income and loan amount requests.

The most common loan repayment period is 360 months.
